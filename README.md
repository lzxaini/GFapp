# 光馥科美 · 微信小程序

✨「光馥科美」是一款基于微信生态开发的小程序，旨在提供高效、便捷、美观的用户体验。

---

## 📦 项目简介

- **项目名称**：光馥科美
- **技术栈**：微信小程序（原生）
- **主要功能**：
  - 产品浏览与展示
  - 用户注册与登录
  - 控制设备功能
  - 后台管理接口对接
  - ……
